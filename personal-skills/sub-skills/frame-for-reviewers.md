# Frame for Reviewers

How Nicole writes for reviewers and reviewing peers.

## Structure

1. **Lead with the thesis.** "This is intentional." / "Yes, by design." / "I disagree, here's why."
2. **State the reasoning** in one or two sentences. Concrete, not abstract.
3. **Acknowledge the broader concern** without conceding the specific point. ("Happy to discuss as a team if there are broader concerns about how these files are managed going forward.")

## Language Calibration

| Replace... | With... |
|---|---|
| "I feel like..." | "This is..." / "I think..." |
| "I was wondering if..." | "Worth considering..." |
| "Maybe we could..." | "We could..." |
| "Sorry, but..." | (delete) |
| "Just a thought" | (delete) |

Confident, not aggressive. Warm, not obsequious.

## Tone Test

Before sending, ask:
- Does it lead with what I actually think?
- Does it land the point without softening it past the point of clarity?
- Would I respect this response if I received it?

If yes to all three, ship it.
